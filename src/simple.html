What This Demonstrates

  • Progressive Loading: main.js (~3KB) loads immediately, components load on interaction
  • Vanilla JavaScript: No frameworks - pure DOM manipulation
  • Small Bundles: Each component is 1-2KB of focused functionality
  • Fast Interactions: Components initialize instantly once loaded
  • Network Efficiency: Only loads what users actually interact with

Network Loading Pattern

  1. Page Load: HTML, CSS, and main.js (~3KB) load immediately
  2. First Interaction: Component-specific JS loads dynamically (~1-2KB)
  3. Subsequent Interactions: Already loaded components respond instantly

Check your Network tab: You'll see individual component files load only when you interact with them!
